PCC Script Directory

Drag the blue buttons to your Bookmarks Bar. Click them while on the CMS to activate the script.

  • Job Dashboard

    This script is provides a full featured dashboard for active jobs including links to XActanalysis if the job is an assignment.

    1. Open the Job List Report
    2. Make sure you are on page 1
    3. Click the bookmarklet and let the next page load
    4. Repeat step 3 until all pages have been processed and the report renders
    Simplify Jobs
  • Rich Text Notes

    This script adds functionality for formatted notes and note templates.

    1. Open a Job's notes and click "+ Add New"
    2. Once the note editor is visible, click the bookmarklet
    Rich Text Notes
  • Estimate Kanban

    This script is for estimators to visually track the progress of an estimate.

    1. Open the Estimate Kanban Report
    2. Make sure you are on page 1
    3. Click the bookmarklet and let the next page load
    4. Repeat step 3 until all pages have been processed and the report renders
    Estimate Kanban
  • WO to Calendar

    This script displays work orders in a calendar view and provides a download link so that the calendar can be imported into an external calendar app.

    1. Navigate to the Work Orders page
    2. Make sure you are on the first page
    3. Click the bookmarklet and let the next page load
    4. Repeat step 3 until all pages are processed and the calendar renders
    WO to Calendar
  • Estimate Discrepancy Filter

    This script is for accounting to highlight which jobs have estimate amounts and invoice amounts that do not match.

    1. Open the Job List Report
    2. Make sure you are on page 1
    3. Click the bookmarklet and let the next page load
    4. Repeat step 3 until all pages have been processed and the report renders
    Estimate Filter
